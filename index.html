    03. Проекты

    Главный проект

    Jarvis AI Assistant

    Комплексный ИИ-ассистент для ПК и умного дома, разработанный с использованием React, Flask и Python. Система интегрирует различные ИИ модели для обработки естественного языка, управления устройствами умного дома и автоматизации повседневных задач. Проект демонстрирует глубокую интеграцию различных технологий и создание удобного пользовательского интерфейса.

    • React
    • Flask
    • Python
    • Искусственный интеллект
    • Web Development

    Веб-приложение

    Premium Business Center

    Современный веб-сайт для премиального бизнес-центра, расположенного в самом сердце города. Проект включает интерактивную систему бронирования помещений, виртуальные туры по офисам, интеграцию с CRM-системой и автоматизацию процессов аренды. Особое внимание уделено премиальному дизайну и удобству использования для потенциальных клиентов.

    • React
    • Node.js
    • MongoDB
    • Three.js
    • Stripe API

    ИИ Проект

    Auto Parts AI

    Система искусственного интеллекта для автоматизации подбора автозапчастей. Использует компьютерное зрение и машинное обучение для идентификации деталей и их совместимости. Проект значительно ускорил процесс подбора запчастей и уменьшил количество ошибок при заказе.

    • Python
    • TensorFlow
    • OpenCV
    • Flask
    • Computer Vision
